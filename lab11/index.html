Challenges

Use text boxes and buttons to input information to js and display output on site.

Problems

I think the most confusing part about the text box and buttons was figuring out when to use the id for the input box and when to use the id of the button. I kept on mixing them up and sometimes I ended up forgeting to put a # before it. I also ended up doing task X, and for that I used animation in CSS to make the background change a bit more. Figuring that out was a bit of a process as well.

Results

Enter words into text box and press submit. If you want another anagram for the same words, press submit again and you'll get a different response. Party mode changes the look of the page for 15 seconds before returning to normal.
