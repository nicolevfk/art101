Lab 2 - HTML & CSS Starter

Challenge

In this lab we had to edit HTML and CSS files in order to make an introductory page for ourselves and our partner.

Problems

We struggled to figure out how to work together while working on different files. In the end, rather than creating the exact same project, we both worked individually on our own projects and helped each other out whenever we got stuck on something. Some examples of this were us figuring out how to replicate the formatting style already existing on the HTML file, figuring out how to format our paragraphs (including going to a new line with br /), and suggesting ways to find color palettes for the final touch.

Reflection

I enjoyed this project because I got to experiment a bit with HTML and CSS when I actually knew what I was doing. I took another class sophomore year where, during one project, I had to use a software that used HTML/CSS/JS, but I had no idea what I was doing so the result was pretty basic. A large portion of the class were CSGD majors, so I felt a bit out of place as an AGPM major. From a beginners perspective, it can look a bit scary, but HTML and CSS isn't that complicated once you learn a bit about it (at least compared to some of the other software I've had to learn).

Results

I like my final result. I added a png of a planet because that's the vibe I was getting from my color palette. I would have liked to put it next to my text rather rather than separating it, but I haven't figured out how to do that yet. Some of my favorite things I learned were how to indent and align text. I'm not a fan of when text is all aligned to the left because I think it looks boring, so I wanted to change that on my page.
