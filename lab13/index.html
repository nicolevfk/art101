Problems

I had a little trouble figuring out how I would set up a function to get the input from all the input boxes at the same time and how to store them to use in the FizzBuzz function. I looked at Wes' code to see how he designed this part and ended up making my code very similar. I also wanted to make it so that I could change the inputs and it would update when I hit the button again, but quickly figured out that because I was using append to add onto the input, instead of changing everything, it was just adding the new output onto the end of the old output. To fix this I made it so during the first for loop (when n=0) the output is overwritten with .html().
